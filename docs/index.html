Article Cloner

LP記事をスクレイプ → ブロック編集 → AI画像生成 → Squad Beyond互換HTMLに変換。
フルWeb UIでブラウザ上で完結。

# インストール & 起動
cd article-cloner
npm install
npm start

# ブラウザで開く
http://localhost:3000

URLを入力するだけで自動スクレイプ → ブロック分割 → プレビュー表示。
各ブロックをクリックして編集、テキスト一括差し替え、AI画像生成、SB互換HTML出力まで全てUI上で操作可能。
