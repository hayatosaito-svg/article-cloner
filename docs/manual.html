概要・全体フロー

Article Clonerは、競合アフィリエイトLPを自動でスクレイピングし、テキスト・画像・リンクを差し替えて、Squad Beyond互換のHTMLを出力するツールです。

STEP 1: LPスクレイピング

手順
競合LPをスクレイピングする
  1. トップ画面のURL入力欄に競合LPのURLを貼り付ける
  2. 「クローン開始」ボタンをクリック
  3. 自動でページを取得 → 画像・テキスト・HTML構造を解析
  4. 完了するとエディター画面に自動遷移

スクレイピング対象

要素取得内容
画像全ての img / picture / source 要素（WebP, AVIF, PNG, JPG）
動画video要素のMP4ソース
テキスト全テキスト要素（見出し、本文、CTA等）
CSSインラインstyle + styleブロック
リンク全てのaタグのhref
Squad Beyond上のLPをスクレイピングする場合は、プレビューURL（sb-draft-preview...）を使用してください。公開URLでも動作しますが、プレビューURLの方がSB固有のウィジェット構造を正確に取得できます。
スクレイピングには10〜30秒かかります。ページが重い場合やアセットが多い場合はさらに時間がかかることがあります。処理中はブラウザを閉じないでください。

STEP 2: リンク置換

手順
LP内のリンクを一括差し替え
  1. 左パネルの「リンク置換」セクションを開く（▶ をクリック）
  2. LP内の全リンクが一覧表示される
  3. 差し替えたいリンクにチェックを入れる（「全て選択」も可能）
  4. 下部の入力欄に新しいURLを入力
  5. 「置換」ボタンで一括差し替え
「リンクを別タブで開く」にチェックを入れると、全てのリンクに target="_blank" が追加されます。

STEP 3: 画像差し替え

AI画像生成で差し替え

  1. エディター右側のプレビューで画像ブロックをクリック
  2. 右パネルに「AI編集」が表示される
  3. 「AIプロバイダーを選択」でPixAI or Nano Banana Proを選択
  4. プロンプトを入力して「生成」
  5. 生成結果をプレビューし「適用」
プロバイダー特徴おすすめ用途
PixAIアニメ・イラスト系に強い漫画LP、キャラクター
Nano Banana Pro写真・リアル系に強い施術写真、ビフォーアフター、バナー

画像アップロードで差し替え

  1. 外部ツール（Leonardo.ai / Canva等）で画像を作成
  2. 画像ブロックをクリック → 右パネル「AI編集」
  3. 「画像アップロード」エリアに画像をドラッグ or クリックで選択
  4. アップロード後、自動でブロックに適用

漫画LP制作フロー

漫画形式のLPはストーリーで商品の魅力を伝える手法です。PixAI → Canva → アップロードの3ステップで制作します。

1. PixAI でキャラ作成
→
2. Canva で吹き出し作成
→
3. 画像アップロード

PixAI でキャラクター作成

  1. PixAI にログイン → 「Generate」をクリック
  2. モデルを選択（おすすめ: Anything V5 / Counterfeit）
  3. プロンプトを入力してキャラクター画像を生成
  4. サイズ: LP用は 512x768（縦長）or 768x512（横長）

プロンプト例

シーンプロンプト
悩む女性1girl, japanese, 30s, worried expression, looking at mirror, manga style, simple background
笑顔の女性1girl, japanese, 30s, happy smile, beautiful skin, manga style, sparkle effect
医師キャラ1man, japanese doctor, white coat, gentle smile, professional, manga style
同じキャラを使い回すには seed値を固定 する。白背景で生成するとCanvaで編集しやすい。

Canva で吹き出し作成

吹き出しテンプレート（すぐ使える）:
Canva テンプレートを開く
「ファイル」→「コピーを作成」で複製してから編集してください

テンプレートを使う方法（推奨）

  1. 吹き出しテンプレート を開く
  2. 右上「ファイル」→「コピーを作成」でテンプレートを複製
  3. PixAIで作った画像をアップロードして差し替え
  4. 吹き出し内のテキストを編集
  5. PNG/JPGでエクスポート → Article Clonerにアップ

ゼロから作る方法

  1. Canva にログイン
  2. 「カスタムサイズ」で 580×任意の高さ（LP幅に合わせる）
  3. PixAIで作った画像をアップロード
  4. 「素材」から「吹き出し」「speech bubble」で検索して配置
  5. テキストツールでセリフを入力

漫画風テクニック

  • コマ割り: 「フレーム」素材で画像を区切る / 四角形で枠線作成
  • 吹き出し: 素材検索「speech bubble」→ サイズ調整してセリフ入力
  • 効果線: 「集中線」「manga lines」で検索
  • フォント: 「M PLUS Rounded」「Kosugi Maru」がおすすめ
  • オノマトペ: 大きく太いテキストで「ドキッ！」「キラーン」を追加

エクスポート設定

  • 形式: PNG（透過不要ならJPG）
  • 品質: 最高画質
  • 1コマずつ別ページにして一括ダウンロードすると管理しやすい

STEP 4: コピー＆書き出し

ボタン用途出力内容
全体コピーGoogleドキュメント・外部エディタに貼り付け画像付きリッチHTML（外部CDN URLで画像表示）
HTMLコピーSquad Beyondに貼り付けSB互換HTML（picture/source/lazyload対応）
画像一覧・コピー個別画像をコピークリックで画像をクリップボードにコピー
全体コピー → Googleドキュメント
画像付きでドキュメントに貼り付け
  1. 左パネルの「全体コピー」ボタンをクリック
  2. 「コピー完了」のアラートが出たら、Googleドキュメントを開く
  3. Cmd+V（Ctrl+V）で貼り付け
  4. 画像が自動で読み込まれて表示されます
HTMLコピー → Squad Beyond
SB互換HTMLで公開
  1. 左パネルの「HTMLコピー」ボタンをクリック
  2. Squad Beyondのエディターを開く
  3. カスタムHTMLウィジェットに貼り付け
  4. プレビューで表示確認 → 公開
